# 趣味动画简历

<a name="NSGuU"></a>
##  游戏界面说明：
<br />   整个游戏的设计基于一个小女孩在春天梦幻的场景中不知疲倦地追逐着背着行李的精灵大猫，太阳和月亮陪随着他们。在追逐过程中也有很多动物动画，充满着春天朝气蓬勃的氛围。

动画简历通过内容设置、建立视差滚动、制作角色外观和常态动画、制作场景对象动画事件搭建起整个动画画面，让简历变得更加生动有趣，里面用到了很多图片元素，还有很多css3动画animation，还有jquery动画animate。
